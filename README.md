# peatlland_vegetation_mapping
Code to replicate main analyses from Mapping peatland plant communities dynamics using multispectral indices coupled with a joint species distribution model (https://www.frontiersin.org/journals/environmental-science/articles/10.3389/fenvs.2026.1863447). 

Input data to run the codes can be find in Zenodo: 10.5281/zenodo.21297992.
